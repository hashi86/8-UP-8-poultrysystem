# إعداد GitHub Pages

## خطوات التفعيل:

1. انتقل إلى إعدادات المستودع (Settings)
2. اختر "Pages" من القائمة الجانبية
3. اختر "Deploy from a branch"
4. اختر الفرع `main` والمجلد `/ (root)`
5. انقر "Save"

## الرابط:
سيكون موقعك متاحاً على:
```
https://hashi86.github.io/6-UP-6-poultrysystem/
```

## المميزات:
✅ متجاوب تماماً مع الهواتف والأجهزة اللوحية
✅ يدعم اللغة العربية والإنجليزية
✅ بيانات تجريبية للعرض
✅ لا يتطلب backend server
